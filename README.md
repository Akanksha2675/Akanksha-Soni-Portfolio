## 📌 About

This is my personal developer portfolio — built to showcase my projects, skills, certifications, and achievements. Designed with a dark cyberpunk aesthetic using electric violet, cyan, and magenta accents on a deep space navy background.

The entire site is a **single self-contained HTML file** with no external frameworks.

## 🏗️ Local Setup

Since this is a single HTML file, no installation is needed:

```bash
# Clone the repo
git clone https://github.com/Akanksha2675/Portfolio.git

# Open in browser
open index.html
```

Or just drag `index.html` into any browser.
